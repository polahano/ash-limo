# Ash Limousine & Charter Bus Website

Static multi-page website package for Ash Limousine & Charter Bus.

## Pages included
- Home
- Services
- Fleet
- Airport Transfers
- Corporate Transportation
- Wedding Transportation
- FIFA 2026
- Quote Request
- About
- FAQ
- Terms
- Privacy Policy

## Files
- `index.html` and supporting page templates
- `css/styles.css`
- `js/app.js`
- `assets/` for icons and illustrations
- `robots.txt`
- `sitemap.xml`

## Deployment
Upload the full folder contents to the web server or static hosting provider so the HTML files, CSS, JavaScript, and assets stay together.
